# To-Do List Application

A simple and interactive **To-Do List** web application built with **HTML, CSS, and JavaScript**. Users can add tasks, mark them as completed, and delete tasks.  

## Features

- Add new tasks
- Mark tasks as completed by clicking on them
- Delete individual tasks
- Clean and responsive design
- Lightweight and easy to use
